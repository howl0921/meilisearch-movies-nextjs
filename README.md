# MovieDB - 电影数据库

这是一个使用Next.js构建的电影数据库应用，允许用户搜索电影、查看详细信息并管理观看列表。

## 功能特性

- 电影搜索功能
- 电影详情查看
- 观看列表管理
- 响应式设计
- 暗色主题

## 技术栈

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Lucide React Icons

## 环境要求

- Node.js 18+
- pnpm

## 安装步骤

1. 克隆项目仓库
2. 安装依赖：

```bash
pnpm install
```

3. 配置环境变量（参考 `.env.example` 文件）
4. 启动开发服务器：

```bash
pnpm dev
```

5. 在浏览器中打开 [http://localhost:3000](http://localhost:3000)

## 环境变量配置

请参考 `.env.example` 文件配置必要的环境变量。

## 部署

可以使用 Vercel 或其他支持 Next.js 的平台进行部署。
